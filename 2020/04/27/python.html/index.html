opencv读图片的问题

#中文目录的话用imread方法会出现问题。
#解决方案：
def readimg(filename, mode):
    raw_data = np.fromfile(filename, dtype=np.uint8)  #先用numpy把图片文件存入内存：raw_data，把图片数据看做是纯字节数据
    img = cv2.imdecode(raw_data, mode)  #从内存数据读入图片，mode可以是cv2.IMREAD_COLOR（表示彩色图片，读进来三个通道）
    return img
#opencv读进来的是[宽,高,通道数]
