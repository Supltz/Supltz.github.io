脉冲神经网络的学习

环境要求

  • PyTorch 1.5
  • Anaconda 3
  • python 3.6.5
  • CUDA 10.2
  • Visual Studio 2017
    只能在支持CUDA的GPU（最好是N卡）上运行，暂不支持在CPU上运行

项目部署

项目内有C++和CUDA的代码需要先把他们编译才能为python所用，进入项目目录执行：

python setup.py install

测试安装是否成功的代码:

cd test

python -m unittest

windows10 下会出现的问题

问题一

解决办法：
1.在 项目内的argument_spec.h文件中,
用 static const size_t DEPTH_LIMIT = 128; 代替 static constexpr size_t DEPTH_LIMIT = 128;
2.在 项目内的module.h中,
用 static const bool all_slots = false; 代替所有的 static constexpr bool all_slots = false;
3.在项目内的cast.h文件中,
用 explicit operator type& () { return *((type*)(this->value)); } 代替 explicit operator type&() { return *(this->value); }

解决办法

在 src/slayer.py文件中, 找到101行和112行把torch.tensor改成torch.FloatTensor

或者尝试：

在slayer.py文件中，把torch.tensor(srmKernel) and torch.tensor(refKernel)改成torch.tensor(srmKernel).float() and torch.tensor(refKernel).float()
